# RSS-TUI

An RSS feed reader inside your terminal, retrieves data from multiples feeds to get the freshest news in a minimalistic interface

## Table of content
- Features
- Installation
- Usage
- Technologies used
- Contributing

### Features
Manage feeds (add/remove/list)
Manual refresh
Unread/read tracking
Keyboard-driven TUI navigation
Local storage (config + state)

#### Planned
Categories/tags for feeds
Global “Unread” view
Sorting/filtering
OPML import/export
Auto-refresh
